# 🛡️ NEXSEC 2025 CTF Writeup  
**Team:** BATERIAAA  

This repository contains our full write-up for the NEXSEC 2025 Capture The Flag (CTF) competition.  
The challenges cover multiple cybersecurity domains including reverse engineering, malware analysis, digital forensics, and incident response.

---

## 📌 Overview
During this competition, our team analyzed real-world inspired attack scenarios such as:
- Malware infections
- Reverse shells & persistence
- Ransomware behavior
- Disk & memory forensics
- Web exploitation

We documented each solution step-by-step to demonstrate both methodology and tools used.

- Reverse Engineering (Binary & DLL analysis)
